APAN Federation Demonstration

This site demonstrate the following identity federation features.

  • Federeated logout or Single Logout (SLO)
  • Multi Factor Authentication

      • Login Login to the site using whatever Authentication Factor is available at the IdP from the following list. The list is in preference order, the IdP will authenticate the user with first available supported authentication method.
      • REFEDS MFA Profile - https://refeds.org/profile/mfa
      • REFEDS Single Factor Authentication Profile - https://refeds.org/profile/sfa
      • PasswordProtectedTransport - urn:oasis:names:tc:SAML:2.0:ac:classes:PasswordProtectedTransport
      • Password - urn:oasis:names:tc:SAML:2.0:ac:classes:Password
      Login MFA - Option 1 Login with MFA is required. If MFA is not provided access to the site is denied
      Login MFA - Option 2 Login with MFA is required. If MFA is not provided a error state will occur and access will be denied
